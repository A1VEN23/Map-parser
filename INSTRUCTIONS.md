# 🚀 Инструкция по настройке Lead Finder

## 📋 Обзор
**Lead Finder** - бесплатный инструмент для поиска потенциальных клиентов из Google Maps с умной фильтрацией "болей" бизнеса.

## 🔧 Установка и настройка

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Получение API ключа Google Maps (БЕСПЛАТНО)

#### Шаг 1: Создание проекта в Google Cloud
1. Перейдите в [Google Cloud Console](https://console.cloud.google.com/)
2. Нажмите "Создать проект"
3. Введите название проекта (например: "Lead Finder")
4. Выберите существующий billing account или создайте новый

#### Шаг 2: Включение API
1. В созданном проекте перейдите в "API и сервисы" → "Библиотека"
2. Найдите и включите следующие API:
   - **Places API**
   - **Geocoding API**
   - **Maps JavaScript API** (опционально)

#### Шаг 3: Создание API ключа
1. Перейдите в "API и сервисы" → "Учетные данные"
2. Нажмите "Создать учетные данные" → "API ключ"
3. Скопируйте созданный ключ

#### Шаг 4: Ограничение API ключа (ВАЖНО!)
1. В настройках API ключа установите ограничения:
   - **Ограничения приложений**: Нет (для локального использования)
   - **Ограничения API**: Places API, Geocoding API
2. Это защитит ваш ключ от несанкционированного использования

### 3. Настройка переменных окружения

#### Windows:
```cmd
set GOOGLE_PLACES_API_KEY=ваш_api_ключ
```

#### PowerShell:
```powershell
$env:GOOGLE_PLACES_API_KEY="ваш_api_ключ"
```

#### Для постоянной настройки:
1. Нажмите Win + R, введите `sysdm.cpl`
2. Перейдите в "Дополнительно" → "Переменные среды"
3. Создайте новую системную переменную:
   - Имя: `GOOGLE_PLACES_API_KEY`
   - Значение: ваш_api_ключ

## 💰 Бюджет и лимиты

### Бесплатный лимит Google Cloud
- **$200 в месяц** бесплатно для новых пользователей
- **$300 кредит** при регистрации нового аккаунта
- Достаточно для ~50,000 запросов к Places API

### Стоимость запросов
- Places Text Search: ~$0.032 за 1000 запросов
- Place Details: ~$0.017 за 1000 запросов
- При поиске в 5 городах по 3 категориям: ~300 запросов = ~$0.01

## 🎯 Использование

### Запуск скрипта:
```bash
python lead_finder.py
```

### Результаты:
- Файл `leads.csv` со всеми найденными лидами
- Автоматическая фильтрация "горячих" лидов
- Статистика по "болям" бизнеса

## 🔥 Критерии "горячих" лидов

### Автоматическая пометка как ГОРЯЧИЙ ЛИД:
1. **Рейтинг ниже 4.3** - плохой сервис
2. **Менее 10 отзывов** - не занимаются продвижением  
3. **Отсутствие чат-бота** на сайте
4. **Нет формы онлайн-записи** (для стоматологий/автосервисов)
5. **Нет контактной формы** на сайте
6. **Полное отсутствие сайта**

## 🔧 Интеграция с ИИ-звонарем

Файл `leads.csv` готов для автоматического импорта в ИИ-звонарь:
- `is_hot_lead=TRUE` - приоритетные клиенты
- `pain_points` - готовые аргументы для звонка
- `phone` - прямые номера для обзвона

## 🚨 Важные замечания

1. **Соблюдайте лимиты API** - не более 100 запросов в минуту
2. **Используйте прокси** для больших объемов
3. **Резервируйте API ключ** - создайте отдельный ключ для продакшена
4. **Следите за расходами** в Google Cloud Console
